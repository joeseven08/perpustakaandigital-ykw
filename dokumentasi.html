Pembuatan Website Perpustakaan Digital YKW

Pembuatan http://perpustakaandigital-ykw.test/ guna memenuhi tugas UAS Pemrograman Web II UNSIA Semester Genap 2025.

Langkah-langkah

  • Pastikan XAMPP sudah terinstall pada laptop.
  • Unduh CodeIgniter versi 3 dan ekstrak folder tersebut ke folder htdocs dalam direktori XAMPP.
  • Beri nama folder yang diekstrak tersebut sesuai nama website.
  • Lakukan konfigurasi yang diperlukan.

Berikut Tampilan Website Perpustakaan Digital

1. Halaman Login

Menampilkan form login untuk pengguna masuk ke sistem.

2. Error Login

Error akan muncul jika username atau password salah.

3. Menu Pencarian

Memungkinkan pengguna mencari buku berdasarkan judul atau pengarang.

4. Menu Tambah Buku

Form untuk menambahkan data buku baru ke dalam database.

5. Menu Edit Buku

Memungkinkan pengguna untuk memperbarui informasi buku yang sudah ada.

6. Tampilan Setelah Pencarian Buku

Menampilkan hasil pencarian buku dalam bentuk tabel.

7. Tampilan Saat Klik Hapus

Menampilkan konfirmasi penghapusan dan aksi penghapusan buku.
